# Viewer V1 Feature Lock (Read-Only)

## Global
- Read-only viewer
- CAD edit: not available
- Export back-to-CAD: not available

## 3D Viewer (GLB/GLTF)
### Included (V1)
- Orbit / Pan / Zoom
- Perspective / Orthographic toggle
- Render modes: Shaded, Wireframe, Hidden Line (basic)
- Model tree (assembly/parts)
- Select + highlight
- Hide/Show part
- Section plane: single plane + slider
- Measure: 2-point distance
- Fullscreen
- Screenshot (PNG)

### Excluded (V1)
- Multi-section
- Advanced measurement suite
- Explode advanced (V2)
- CAD edit/export

## 2D Viewer (PDF/SVG + DXF server-side)
### Included (V1)
- Pan / Zoom
- PDF page select
- Layers toggle (SVG/DXF -> SVG)
- Fullscreen

### Excluded (V1)
- Edit
- DWG
- Browser-side DXF parse
